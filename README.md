## Project Overview

This project demonstrates packet flow monitoring, cybersecurity traffic analysis, and network visualization using Cisco Packet Tracer.

The project focuses on understanding how packets travel across networks, identifying communication patterns, analyzing protocol behavior, and visualizing cybersecurity-related network traffic.

---

## Internship Context

This project was developed during the **Cisco Networking Academy Virtual Internship Program (VIP) 2023** under **NIIT Foundation**.

The objective was to visualize packet flow from host systems inside a college network to external networks using Cisco Packet Tracer and packet analysis concepts.

---

## Original Problem Statement

The objective of this project was to create a visual representation of packet flow from host devices inside a college network to external networks using Cisco Packet Tracer and packet analysis concepts.

The project involved:

- Creating network topology
- Configuring routers and switches
- Assigning IPv4 addresses
- Capturing packets
- Analyzing Ethernet and IP headers
- Understanding OSI layer communication
- Visualizing packet movement across the network

## Network Flow Architecture

This project demonstrates packet traversal through:

```text
Host Device → Switch → Router → External Network
```

The simulation includes:

- Layer 2 Switching
- Layer 3 Routing
- Packet Header Analysis
- OSI Layer Communication
- IPv4 Configuration
- Network Segmentation
